# Fix Compilation Errors

## Overview
The project has several compilation errors that need to be fixed. Let me analyze and plan the solutions.

## Error Analysis

### 1. CharacterData.cs - BindableProperty Operators
- **Error**: `Operator '+=' cannot be applied to operands of type 'int' and 'BindableProperty<int>'`
- **Location**: Lines 64, 75, 88, 99, 112, 123
- **Root Cause**: The code is likely trying to use `+=` and `-=` operators directly on `BindableProperty<int>` objects instead of their `.Value` properties

### 2. CharacterController.cs - Missing Variable
- **Error**: `The name '_characterController' does not exist in the current context`
- **Location**: Lines 151, 153
- **Root Cause**: The variable was renamed to `_unityCharacterController` to avoid conflict, but references to the old name still exist

### 3. XMLDataLoader.cs - Missing Property
- **Error**: `'MapData' does not contain a definition for 'Id'`
- **Location**: Line 260
- **Root Cause**: The code is trying to access `mapData.Id` instead of the correct `mapData.MapId` property

## Fix Plan

- **Action**: Ensure all `BindableProperty<int>` operations use the `.Value` property
- **Files**: `Assets\Scripts\Models\CharacterData.cs`
- **Changes**:

  - Replace `Attack -=` with `Attack.Value -=`
  - Replace `Defense +=` with `Defense.Value +=`
  - Replace `Defense -=` with `Defense.Value -=`

### 2. Fix CharacterController.cs
- **Action**: Replace all occurrences of `_characterController` with `_unityCharacterController`
- **Files**: `Assets\Scripts\Character\CharacterController.cs`

- **Files**: `Assets\Scripts\DataLoaders\XMLDataLoader.cs`

After implementing these fixes, all compilation errors should be resolved, and the project should build successfully.
